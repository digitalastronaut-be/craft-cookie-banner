# Installation and setup

## installing using composer

Install the latest version or a specific realese of the author toolbar via composer with the following console command

::: code-group

```bash[DDEV]
ddev composer install digitalastronaut/craft-cookie-banner
```

```bash[Other]
composer install digitalastronaut/craft-cookie-banner
```

:::

## Installing via the Craft plugin store

Alternatively you can install the cookie banner plugin via the [craft plugin store](https://plugins.craftcms.com/cookie-banner?craft5).
