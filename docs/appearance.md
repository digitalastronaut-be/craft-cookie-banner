# Appearance

You can customize how the cookie banner looks using the appearance page in the control panel or by using custom css properties in your stylesheets as follows:

```css
#ccb-cookie-banner {
	--banner-border-radius: ;
	--banner-background: ;
	--title-color: ;
	--text-color: ;

	--link-color: ;
	--link-hover-color: ;

	--button-background: ;
	--button-color: ;
	--button-border-radius: ;
	--button-border: ;
	--button-hover-background: ;
	--button-hover-color: ;
	--button-hover-boder: ;

	--toggle-border-radius: ;
	--toggle-background-color-off: ;
	--toggle-background-color-on: ;
	--toggle-color: ;

	--cookie-group-background: ;
	--cookie-group-border: ;
	--cookie-group-hover-background: ;
	--cookie-group-hover-border: ;
	--cookie-group-title-color: ;
	--cookie-group-text-color: ;
	--cookie-group-link-color: ;

	--cookie-table-background: ;
	--cookie-table-border: ;
	--cookie-table-border-radius: ;
	--cookie-table-title-color: ;
	--cookie-table-text-color: ;
}
```

> If you want to change other properties or elements you can but keep compliancy in mind.
